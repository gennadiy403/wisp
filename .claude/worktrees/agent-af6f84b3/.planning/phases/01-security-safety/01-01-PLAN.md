| Boundary | Description |

| User config -> subprocess | Editor path from $EDITOR env var passed to subprocess.run |
| User config -> cli_setup() | Language choice selects string set |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Elevation of Privilege | os.system(f"{editor} {path}") | mitigate | Replace with subprocess.run([editor, str(path)]) -- list-form bypasses shell entirely (SEC-01) |
| T-01-02 | Information Disclosure | Privacy notice content | accept | Notice is informational, contains no secrets. ANSI codes are display-only |
| T-01-03 | Tampering | $EDITOR env var could point to malicious binary | accept | User controls their own env; subprocess.run with list args prevents shell expansion of the path argument. The editor binary itself is trusted as user-chosen |
</threat_model>
